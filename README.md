# RunBuddy Process-Portfolio Website
## Project Overview
This document describes the Process-Portfolio website of **RunBuddy**, a gamified companion running application.
The website serves as the official external display platform, fully presenting core features, product highlights and brand concepts of RunBuddy. It aims to help beginner runners develop long-term exercise habits through gamification, character raising, professional running guidance and interactive community. You can access our static web page by [RunBuddy-Process-Portfolio](https://adam-aiaiai.github.io/process-portfolio/)

## Product Introduction
RunBuddy is a gamified companion running application focusing on beginner runners.
We combine **character raising system** with traditional running functions to solve the problem of low persistence for new runners. Meanwhile, we build an active running community for users to communicate and share experiences. Professional running tutorial videos are provided to help users master standard posture and exercise scientifically.

## Core Website Highlights
- Brand display: Show RunBuddy’s positioning, design style and original running companion concept
- Feature introduction: Clearly demonstrate gamified raising, running data recording, community interaction and professional guidance
- User-oriented: Concise layout, intuitive information and friendly browsing experience
- Responsive design: Adapt to desktop, tablet and mobile devices

## Tech Stack
- Frontend: HTML, CSS, JavaScript
